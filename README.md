# Inland water heat uptake calculations
Forked from data processing and analysis used in Vanderkelen et al., 2020.


## For users
This repository is based on the processing and plotting scripts used in Vanderkelen et al, 2020, and elaborated for the Global Heat Inventory in Cuesta-Valero et al., 2023 and von Schuckman et al., 2023. 

The data analysis and plotting is done with python, checkout the [main](main_lakeheat.py) script for an overview. 

The HYDROLAKES and GRanD data is processed using the tool [polygon_to_cellareafraction](https://github.com/VUB-HYDR/polygon_to_cellareafraction).
